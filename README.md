🚀 VerveReply AI

VerveReply AI is an intelligent email reply assistant that generates professional responses using Generative AI.
It integrates a Spring Boot backend with a Chrome Extension to provide real-time email suggestions inside Gmail.

✨ Features

🤖 AI-powered email reply generation

⚡ Real-time integration with Gmail

🎯 Tone-based responses (formal, friendly, etc.)

🔗 Seamless backend–frontend communication

🌐 REST API built with Spring Boot

🛠️ Tech Stack

Backend:

Java 21

Spring Boot

WebClient (Reactive API calls)

Gemini API (Google Generative AI)

Frontend (Extension):

JavaScript

Chrome Extension (Manifest v3)

HTML & CSS
